A.4 Carnegie Mellon University

Carnegie Mellon provides a public-domain implementation called Mindy, which stands for "Mindy Is Not Dylan Yet." Mindy is an experimental byte-compiler and interpreter written in C. Mindy implementations are available for UNIX, WindowsNT, OS/2, and the Macintosh (both 68000 and PowerPC). Harlequin's Dylan Web site provides current links to Carnegie Mellon's Gwydion and Dylan Web site.

Carnegie Mellon's main Web site is located at

http://www.cmu.edu/

Carnegie Mellon's electronic mail address for Gwydion is

gwydion-group@cs.cmu.edu
