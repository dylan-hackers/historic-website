13.3 Source code, modules, and libraries

How is Dylan source code associated with modules and libraries? In Sections 13.1 and 13.2, we looked at a Dylan program from the top down: A program contains libraries; a library contains modules; and a module contains variables and their definitions. We now look at a program from the bottom up, to see how source code is organized into modules, modules into libraries, and libraries into
programs.
