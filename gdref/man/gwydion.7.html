NAME

Gwydion Dylan - A collection of Dylan development tools  

DESCRIPTION

Gwydion Dylan is a collection of free software Dylan development tools for Unix and GNU/Linux systems. It includes the d2c(1) compiler, the mindy(1) interpreter, the melange(1) interface generator and a number of other useful programs. Gwydion Dylan was developed by the Gwydion Group at CMU with DARPA funding, and is now maintained by a group of volunteers.

d2c generates fairly good code--current development versions are about half as good as gcc in this respect. (In fact, Gwydion Dylan uses gcc to generate machine code, but not all of the high-level language features are fully optimized.) The compiler and standard libraries are actually very stable, but in need more work before they're suitable for widespread use.

Unfortunately, the compiler still runs slowly. If this is a problem, you can use Mindy, the included Dylan interpreter, to develop and debug code, and use the compiler only when you need performance.  

SEE ALSO

dylan(7), d2c(1), melange(1), mindy(1), mindycomp(1), make-dylan-app(1), platforms.descr(4).

Much more extensive documentation in various formats is available through the Gwydion Dylan web site at http://www.gwydiondylan.org/.
