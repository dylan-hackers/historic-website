NAME

make-dylan-app - Create a new Dylan program  

SYNOPSIS

make-dylan-app appname  

WARNING

This man page is intended to supplement the main Gwydion Dylan documentation at http://www.gwydiondylan.org/. It provides an incomplete reference of the available options. For a full reference or a tutorial, please see the web site.  

DESCRIPTION

make-dylan-app creates a directory appname containing a minimal Dylan application that can be compiled with d2c(1) or interpreted with mindy(1). This makes it unnecessary to understand the Dylan ``library interchange definition'' just to compile a simple program.

The generated program includes three files: appname.lid, appname-exports.dylan and appname.dylan. The first file describes the program in fashion allegedly understood by different Dylan environments, the second contains library import and export declarations, and the third contains actual source code.

An effort has been made to include libraries suitable for writing small, command-line applications.  

SEE ALSO

gwydion(7), d2c(1), mindy(1).
