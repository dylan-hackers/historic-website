NAME

platforms.descr - Platform descriptions for Gwydion Dylan  

DESCRIPTION

This file contains contains descriptions of each platform supported by d2c(1). Each platform consists of a number of key-value pairs, and platforms may inherit from other platforms.

Several of the Perl scripts used to build Gwydion Dylan also look in this file, but Mindy does not.  

SEE ALSO

gwydion(7), d2c(1), mindy(1).

Much more extensive documentation in various formats is available through the Gwydion Dylan web site at http://www.gwydiondylan.org/.
