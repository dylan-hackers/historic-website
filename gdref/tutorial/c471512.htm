A Sample Set of Libraries

Figure 6-1 and Table 6-1 show the libraries included with mindy, a bytecode compiler produced as part of the Gwydion Project . Also shown are the dependencies between the libraries and the major modules contained within each. For more information on these libraries and their use, see the mindy documentation and source code.
