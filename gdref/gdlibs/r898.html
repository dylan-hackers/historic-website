<file-stream>

Name

<file-stream> — Open abstract class

Summary

The class of streams over disk files.

Superclasses

<positionable-stream>, <buffered-stream>

Init-keywords

locator:

direction:

if-exists:

if-does-not-exist:

Description

When you instantiate this class, an implementation-dependent, indirect instance of it is created. The file being streamed over is opened immediately upon creating the stream.
